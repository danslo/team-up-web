# TeamUp Web
v1.15.0

## Install
1. npm install
2. bower install
3. Remove the .example from app/scripts/localConfig.js
4. Remove the .example from test/spec/testConfig.js and ask a employee for the details of a test account

## Start dev
1. grunt serve

## Start testing
1. grunt test

## Create production build
1. grunt build

## Start production build
1. grunt build
